# Examples

The examples live in the docs rather than in a separate root-level `examples/` folder.
Each example page renders the actual ANSI terminal chart during the docs build, so the output stays in sync with the code.

- [Line Charts](basic.md)
- [Scatter Plots](scatter.md)
- [Reference Lines](reference_lines.md)
- [Stacked Bars](stacked_bars.md)

Layout and axis behavior:

- [Dual Axes](dual_axes.md)
- [Linked Layouts](linked_layouts.md)

Use `display(fig)` in the REPL for the normal interactive workflow.
